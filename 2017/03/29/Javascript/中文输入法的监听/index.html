中文输入法监听

发表于 2017-03-29 | 分类于 Javscript |

原因

当在input输入框中输入中文与英文输入不同，英文输入的同时会在输入框中展示，而输入中文则需要等待候选词确定后，输入框的值才是正确的值。

实现

  • compositionstart

当浏览器有非直接的文字输入时, compositionstart事件会以同步模式触发

  • compositionend

当浏览器是直接的文字输入时, compositionend会以同步模式触发.
